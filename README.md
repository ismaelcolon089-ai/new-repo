# GitHub Demo Lab

This repository was created to complete a Git and GitHub lab on basic version control workflows.

## What this repo demonstrates
- Initializing a local Git repository
- Creating and tracking files
- Staging and committing changes
- Creating and working in a branch
- Merging branch changes back into `main`
- Pushing a local repository to GitHub
- Pulling and maintaining remote project content

## Files
- `newfile.txt` — sample file used for staging, commits, and branch changes
- `README.md` — repository overview and course reflection

## Course reflection
My hope for this course is to become more confident using Git and GitHub, improve my workflow, and keep building stronger coding skills.
